# MailParser
Автоматическая система, которая читает входящие обращения из файловой системы, классифицирует их по категориям и формирует отчёт о проделанной работе.

## Выделенные категории

SPAM - спам и фишинг
INCIDENTS - ошибки, сбои, инциденты
REQUEST - запросы, заявки, обращения
FINANCE - все, что связано с финансами: счета, договора, оплата
UNKNOWN - письма, которые не подошли ни в одну из категорий
BROKEN - поврежденные файлы, либо файлы неподходящего формата

## Работа системы
Запускается процессор, который парсит письма из папки в удобный формат и категоризирует их. После категоризации письмо сортируется в нужную папку по категорию.

## Запуск
Из корня приложения:
```bash
python3 -m MailParser -i inbox -o output
```
Параметры:
-i, --input - папка с письмами (по умолчанию inbox)
-o, --output - папка с отсортированными по категориям письмами (по умолчанию output)
-st, --stats - статистика по категориям
-cnt CATEGORY - вывести кол-во писем в конкретной категории

## Запуск через Bash
Скрипт проверяет наличие папки inbox, запускает приложение и выводит статус завершения.

```bash
bash run.sh
```

## Тесты
Тесты запускаются командой:
```bash
python3 -m pytest
```

## Логи
