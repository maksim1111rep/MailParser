# MailParser
Автоматическая система, которая читает входящие обращения из файловой системы, классифицирует их по категориям и формирует отчёт о проделанной работе.

## Выделенные категории

1. SPAM - спам и фишинг
2. INCIDENTS - ошибки, сбои, инциденты
3. REQUEST - запросы, заявки, обращения
4. FINANCE - все, что связано с финансами: счета, договора, оплата
5. UNKNOWN - письма, которые не подошли ни в одну из категорий
6. BROKEN - поврежденные файлы, либо файлы неподходящего формата

## Работа системы
Запускается процессор, который парсит письма из папки в удобный формат и категоризирует их. После категоризации письмо сортируется в нужную папку по категории.

## Запуск
Из корня приложения:
```bash
python3 -m MailParser -i inbox -o output
```
Параметры:
1. -i, --input - папка с письмами (по умолчанию inbox)

2. -o, --output - папка с отсортированными по категориям письмами (по умолчанию output)

3. -st, --stats - статистика по категориям

4. -cnt CATEGORY - вывести кол-во писем в конкретной категории

## Запуск через Bash
Скрипт проверяет наличие папки inbox, запускает приложение и выводит статус завершения.

```bash
bash run.sh
```

## Тесты
Тесты запускаются командой:
```bash
python3 -m pytest
```

## Логи
